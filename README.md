📅 Cronograma de Estudos (20 semanas)

Fase 1 – Fundamentos (Semanas 1–6)
- Lógica de programação  
- Python ou JavaScript  
- Estruturas de dados  
- Exercícios práticos  
- Projetos simples  
- Revisão semanal  

Fase 2 – Desenvolvimento Web (Semanas 7–12)
- HTML e CSS  
- JavaScript avançado  
- Framework front-end (React/Vue)  
- Backend (Node.js ou Python)  
- Projeto web completo  
- Deploy e revisão  

Fase 3 – Inteligência Artificial (Semanas 13–18)
- Fundamentos de Machine Learning  
- Bibliotecas Python (NumPy, Pandas, Scikit-learn)  
- Redes neurais simples (TensorFlow/PyTorch)  
- Mini-projetos  
- Projeto integrador com IA  
- Documentação do portfólio  

Fase 4 – Preparação para o Mercado (Semanas 19–20)
- Currículo e LinkedIn focados em tech  
- Portfólio com projetos publicados  
- Simulação de entrevistas técnicas  
- Networking e comunidades
